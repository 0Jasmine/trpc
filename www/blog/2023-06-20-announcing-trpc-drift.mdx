Today, we're excited to announce a new product we've had in the works for a while - tRPC Drift. Drift seeks to solve common community problems a lot of users have been facing with tRPC:

- Detecting changes to your router: When you make changes to your router, you want to be able to detect if there are any changes that you haven't explicitly approved.

- Generating code from your router: You want to be able to generate code from your router, such as output schemas for validation.
